5 HTML Tags I Learned

<header>
Represents the top section of a page or section, often containing a title and navigation.
<nav>
Contains navigation links that help users move around the site.
<main>
Holds the main content of the page (unique content, not repeated like navigation).
<article>
Represents a complete piece of content that can stand on its own (like a blog post).
<img>
Displays an image using the src attribute and requires alt text for accessibility.
